# ScottAI Roadmap

## Phase 1 – Foundation

- Install and configure n8n
- Connect OpenAI
- Connect to Merlin PostgreSQL
- Prove stock lookup
- Create GitHub repository
- Request dedicated read-only database account

## Phase 2 – Product Search

- Search by F R Scott product code
- Search by manufacturer part number
- Search by alternate part number
- Search by description
- Return multiple possible matches
- Ask the user to clarify where necessary
- Confirm Merlin's official available-stock calculation

## Phase 3 – Customer and Credit Tools

- Customer lookup
- Outstanding balance
- Credit limit
- Available credit
- Account stop status
- Overdue invoices

## Phase 4 – Purchasing Tools

- Outstanding purchase orders
- Expected delivery dates
- Products below minimum stock
- Supplier lead times
- Suggested replenishment
- Excess stock review

## Phase 5 – User Interface

- Improve n8n chat prototype
- Add Microsoft Teams interface
- Add employee authentication
- Add role-based permissions
- Add audit logs

## Phase 6 – Internal AI Platform

- Create reusable ScottAI backend
- Introduce Python or FastAPI where useful
- Add reporting and monitoring
- Add Outlook and Teams integrations
- Add finance, sales and management assistants
